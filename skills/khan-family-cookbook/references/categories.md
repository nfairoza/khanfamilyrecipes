# Categories — auto-sort rules + color registry

## How to sort

Decide from the recipe's **title first, then ingredients/method**. Walk the table
top-to-bottom; first match wins. Every biryani (chicken, mutton, veg, egg) goes under
**Biryani**, not Rice. If genuinely ambiguous, pick the most likely section, say so in
the confirmation list, and let Noor override with one word.

Regional collections (Hyderabadi, Andhra, Korean, Chinese…) become their own category
only when Noor asks for one or when 3+ recipes share the region and don't fit an
existing section.

## Color registry (persistent — always reuse, append new rows here)

A category must get the **same accent every run**. When creating a new category,
assign the next free hue from the palette, add a row here, and keep the file updated.

Free hues remaining in the family palette for new categories:
sunflower variants. Stay inside the bright family — no muted/dusty tones.


## Region/style tags

Add small chips on the recipe banner for Hyderabadi, Andhra, Indo-Chinese, Korean,
etc. Tags don't change the color — only the category does.
